# TypeScript project skeleton

## Features:

- Broccoli + TypeScript + Rollup build setup
- Source maps that work when debugging in Chrome or Visual Studio Code
- vscode launch config for debugger

## Requirements:

- node
- yarn

## Installing:

```bash
$> yarn
```

## Using:

### Development:

```bash
$> yarn start
```

### Production build:

```bash
$> yarn build
```

Builds output to a dist/
